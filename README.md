## Usage:

Run these commands to run WEB-DEFACE

### > For Termux:
**Notice:**
* `pkg install python2 pip2`

* `pkg install git`

* `git clone https://github.com/navin-hariharan/WEB-DEFACE`

* `cd WEB-DEFACE`

* `./start.sh`

### > For Linux:
**Notice:**

* `apt install python2 python-pip`

* `pkg install git`

* `git clone https://github.com/navin-hariharan/WEB-DEFACE`

* `cd WEB-DEFACE`

* `./start.sh`
